# personal portfolio website 
A modern personal portfolio website built with Next.js, React, Tailwind CSS, and Framer Motion.
It showcases personal projects, skills, and contact information in a clean and responsive design.

The project includes:

Dynamic projects section with GitHub and live demo links
Interactive UI with smooth animations
Dark / Light mode support
Responsive design for all devices
3D hero section for a modern visual experience

This portfolio is designed to present developer work in a professional and interactive way.
